Version consultable par l'application Dictionnaire de Mac OS X

  • Télécharger le fichier compressé du dictionnaire;
  • Si le système ne l'a pas déjà fait, décompresser le fichier zip en double-cliquant;
  • Déplacer le fichier LVF.dictionary dans le répertoire ~/Bibliothèque/Dictionaries (le créer s'il n'existe pas);
  • Lancer l'application Dictionnaire, activer le dictionnaire LVF dans les préférences;
  • Il est possible de choisir les champs du LVF à afficher dans les préférences.

Développement de ces formats

Le fichier XML a été produit avec une transformation par une feuille de style XSLT appliquée à la version Excel fournie par Denis Le Pesant du Modyco (Le laboratoire Modèles, Dynamiques, Corpus, UMR 7114 du CNRS/Université de Paris 10). Ces travaux se situent dans le cadre du groupe FondamenTAL.

La transformation a été développée initialement par Patrick Paul, sous la direction de Guy Lapalme, en août et septembre 2008 au laboratoire RALI de l'Université de Montréal.

Cette présentation explique les grandes lignes de la méthodologie de transformation. Ce document présente l'organisation du site et compare les informations du LVF avec celles d'autres sites de ressources lexicales. Plus de détails sont donnés dans cet article publié dans Langages, nos 179-180, p. 193-220, décembre 2010.
